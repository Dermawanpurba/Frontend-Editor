## ✨ Fitur Utama

- 🔍 **Pencarian Code Instan** - Temukan kode dengan cepat menggunakan fitur search real-time
- 🖥️ **Live Server** - Lihat perubahan kode secara langsung tanpa perlu refresh manual
- 📝 **Editor Code Modern** - Interface yang user-friendly dan responsif
- ⚡ **Performa Optimal** - Loading cepat dan proses rendering yang efisien

## 🚀 Instalasi

### Prerequisites

- Node.js (v14 atau lebih tinggi)
- npm atau yarn

### Langkah-langkah

1. Clone repository
```bash
git clone https://github.com/Dermawanpurba/Frontend-Editor.git
cd Frontend-Editor
```

2. Install dependencies
```bash
npm install
```

3. Jalankan aplikasi
```bash
npm start
```

4. Buka browser dan akses `http://localhost:3000`

## 💻 Penggunaan

### Fitur Pencarian
- Gunakan `Ctrl + F` (atau `Cmd + F` pada Mac) untuk membuka fitur pencarian
- Ketik kata kunci yang ingin dicari
- Hasil akan ditampilkan secara real-time

### Live Server
- Setiap perubahan yang dilakukan otomatis tersimpan
- Preview akan update secara otomatis di panel sebelah kanan

## 🛠️ Teknologi

- **Frontend Framework**: React / Vue (sesuaikan dengan stack yang digunakan)
- **Editor**: Monaco Editor / CodeMirror
- **Live Server**: Express.js atau serupa
- **Styling**: CSS / Tailwind CSS (sesuaikan)
